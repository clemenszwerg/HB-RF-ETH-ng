HB-RF-ETH-ng

Next Generation Firmware für die HB-RF-ETH Platine. Modernisiert mit Vue.js WebUI, Home Assistant MQTT Integration und vielen weiteren Verbesserungen.

GitHub Repository Releases & Downloads

Features im Überblick

🌐
Moderne WebUI

Komplett neu geschriebene Weboberfläche basierend auf Vue.js 3, Bootstrap 5 und Vite. Responsiv und benutzerfreundlich mit Dark-Mode Unterstützung.

🏠
Home Assistant MQTT

Nahtlose Integration in Home Assistant via MQTT mit Auto-Discovery. Keine manuelle Konfiguration von Sensoren nötig.

🔄
Einfache Updates

Firmware-Updates bequem über die WebUI als Datei-Upload oder direkt via URL von GitHub installieren.

📊
Erweitertes Monitoring

Umfassende Überwachung von CPU, RAM, Temperatur und mehr. SNMPv2 und MQTT Unterstützung für externe Monitoring-Systeme.

🔒
Verbesserte Sicherheit

Rate Limiting, strikte Passwort-Richtlinien und Input-Validierung schützen vor unbefugtem Zugriff.

🚀
ESP-IDF 5.x

Basierend auf dem aktuellen ESP-IDF 5.x Framework für verbesserte Stabilität, Leistung und Zukunftssicherheit.

Installation

Fertig kompilierte Firmware-Images findest du auf der Releases-Seite bei GitHub.

1. Lade die firmware.bin aus dem aktuellen Release herunter 2. Öffne die WebUI deiner Platine 3. Navigiere zu "Firmware Update" und wähle die Datei aus 4. Klicke auf "Update starten" und warte auf den Neustart
